Un message affiché dans la boîte de dialogue.

##### Utilisation

```js
await dialog.message()
```

##### Exemple

```js title="dialogMessage.js"
const message = await dialog.message();
```

##### Retourne

- **&lt;string&gt;**
            **<code><var>returns</var></code>:**   Le message affiché dans la boîte de dialogue.
