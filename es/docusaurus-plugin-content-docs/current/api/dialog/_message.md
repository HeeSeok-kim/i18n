##### Uso

```js
await dialog.message()
```

##### Ejemplo

```js title="dialogMessage.js"
const message = await dialog.message();
```

##### Devuelve

- **&lt;string&gt;**
            **<code><var>returns</var></code>:**   El mensaje mostrado en el diálogo.
