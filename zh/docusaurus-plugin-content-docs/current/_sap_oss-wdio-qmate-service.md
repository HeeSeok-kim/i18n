Qmate Service 是一个用 TypeScript 构建的自定义 WebdriverIO 服务，用于测试 Web 应用程序和 API。

## 页面
请在[这里](https://sap.github.io/wdio-qmate-service/)查看我们的官方更详细文档。

## 安装
要安装最新版本的服务，只需执行以下命令：
```bash
npm install @sap_oss/wdio-qmate-service@latest
```
