`custom$` 允许您使用通过 `browser.addLocatorStrategy` 声明的自定义策略。
在[选择器文档](../../selectors#custom-selector-strategies)中阅读更多关于自定义选择器策略的信息。

##### 用法

```js
$(selector).custom$(strategyName, strategyArguments)
```

```js title="example.js"
it('should fetch the project title', async () => {
    await browser.url('https://webdriver.io')
    await browser.addLocatorStrategy('myStrat', (selector) => {
        return document.querySelectorAll(selector)
    })

    const header = await browser.custom$('myStrat', 'header')
    const projectTitle = await header.custom$('myStrat', '.projectTitle')

    console.log(projectTitle.getText()) // WEBDRIVER I/O
})
```
