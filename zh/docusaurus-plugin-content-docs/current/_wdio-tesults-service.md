Tesults (https://www.tesults.com) 为 WebdriverIO (https://webdriver.io/) 提供的服务

## Description

使用此服务将 WebdriverIO 测试结果数据报告给 Tesults。

## Installation

`npm install wdio-tesults-service --save`

## Documentation

关于安装、配置和使用的完整文档可在 https://www.tesults.com/docs/wdio 获取。
