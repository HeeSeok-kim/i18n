உரையாடல் பெட்டி நிராகரிக்கப்பட்டவுடன் திரும்புகிறது.

##### பயன்பாடு

```js
await dialog.dismiss()
```

##### உதாரணம்

```js title="dialogDismiss.js"
await dialog.dismiss();
```
