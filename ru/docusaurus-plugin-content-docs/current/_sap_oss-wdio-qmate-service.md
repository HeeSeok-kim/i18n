Qmate Service - это пользовательский сервис WebdriverIO, созданный на TypeScript для тестирования веб-приложений и API.

## Pages
Нашу официальную и более подробную документацию можно найти [здесь](https://sap.github.io/wdio-qmate-service/).

## Installation
Чтобы установить последнюю версию сервиса, просто выполните следующую команду:
```bash
npm install @sap_oss/wdio-qmate-service@latest
```
