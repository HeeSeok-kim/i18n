ダイアログに表示されるメッセージ。

##### 使用方法

```js
await dialog.message()
```

##### 例

```js title="dialogMessage.js"
const message = await dialog.message();
```

##### 戻り値

- **&lt;string&gt;**
            **<code><var>returns</var></code>:**   ダイアログに表示されるメッセージ。
