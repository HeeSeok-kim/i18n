Tesults (https://www.tesults.com) сервіс для WebdriverIO (https://webdriver.io/)

## Description

Звітуйте дані результатів тестів WebdriverIO за допомогою Tesults, використовуючи цей сервіс.

## Installation

`npm install wdio-tesults-service --save`
