Funkcja `customs$$` pozwala używać niestandardowej strategii zadeklarowanej przy użyciu `browser.addLocatorStrategy`.
Więcej informacji o niestandardowych strategiach selektorów znajdziesz w [dokumentacji selektorów](../../selectors#custom-selector-strategies).

##### Użycie

```js
browser.custom$$(strategyName, strategyArguments)
```

##### Przykład

```js title="example.js"
it('should get all the plugin wrapper buttons', async () => {
    await browser.url('https://webdriver.io')
    await browser.addLocatorStrategy('myStrategy', (selector) => {
        return document.querySelectorAll(selector)
    })

    const pluginWrapper = await browser.custom$$('myStrategy', '.pluginWrapper')

    console.log(await pluginWrapper.length) // 4
})
```
