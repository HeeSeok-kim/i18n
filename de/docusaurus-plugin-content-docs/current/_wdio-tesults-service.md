Tesults (https://www.tesults.com) Service für WebdriverIO (https://webdriver.io/)

## Description

Melden Sie WebdriverIO-Testergebnisdaten mit Tesults mit diesem Service.

## Installation

`npm install wdio-tesults-service --save`
