##### Utilizzo

```js
await dialog.message()
```

##### Esempio

```js title="dialogMessage.js"
const message = await dialog.message();
```

##### Restituisce

- **&lt;string&gt;**
            **<code><var>returns</var></code>:**   Il messaggio visualizzato nella finestra di dialogo.
